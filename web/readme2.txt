This component allows you to use text formatted via Bootstrap. Certain functions are provided as parameters on the top level of the Component.
Full functionality is established by enabling the Parse HTML parameter.

Newlines in a text DAT are automatically appended as line breaks. You can read more at https://getbootstrap.com/docs/4.0/content/typography/
<hr />&lt;hr/ &gt; creates a horizontal divider, as above.

<b>&lt;b&gt; and &lt;/b&gt; will make text bold.</b>
